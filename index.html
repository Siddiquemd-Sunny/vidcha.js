vidcha.js – Demo

Simple jQuery plugin for interactive HTML5 video chapters

  • 1. Starting
  • 2. Point1
  • 3. Point2
  • 4. Ending

vidcha.js is a jQuery plugin for creating interactive chapters for HTML5 videos. By clicking on a chapter you jump directly to the appropriate position in the video. Also the current chapter is always highlighted when playing the video.
